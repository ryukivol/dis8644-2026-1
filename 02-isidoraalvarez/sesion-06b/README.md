**Correciones del circuito**

Se conecto un extremo de una resistencia de 100kΩ en el pin 3 del 555 que seria sa salida y en el pin 14 que es la entrada del 4017, el otro extremo se coloco en GRND. Se hixo esto para poder eliminar el  ruido.
No lograbamos hacer que sonara el parlan y nos fuimos dando cuenta que habian cosas mal conetctadas (condesadores- clabes conectados en pins que no eram-chips quemados)

Tambien se corrigio en clases que en el CD 4017 se debia retirar las leds para poder probar el circuito, cuando sacamos las leds logramos hacer sonar el parlante, lo que aun pudimos resolver en clases es la afinacion del sonido por lo que se llego un acuerdo de que cada una probara el como afinar. 

Se usa un capacitor de 100μF para separar el control de volumen del parlante. El lado positivo + se conecta a la pata central del control y el negativo - va directo al parlante.
Esto lo hicimos para filtrar la señal, evitando que las piezas se quemen y quitando los ruidos molestos.
